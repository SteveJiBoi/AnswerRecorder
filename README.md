# 托福刷题网站

一个轻量的静态网站，用于让学生（David / Aaron）在线作答托福真题，并自动批改阅读、听力，导出 Excel。

## 功能

- **任务列表主页**：自动列出所有答案文档对应的任务，可搜索；点击进入答题。
- **答题页**（每个任务）：
  - 顶部选择 **答题者（David / Aaron）** 与 **日期**。
  - **Reading 阅读**：按文档自动生成题目（一般 模块1：20 填空 + 15 选择；模块2：10 填空 + 5 选择）。填空为输入框，选择题为选项；自上而下排列；每题下方有备注框。
  - **Listening 听力**：全部选择题（一般 模块1：32 题；模块2：15 题），自上而下排列，每题有备注框。
  - **Writing 写作**：
    - 模块1 *Make an appropriate sentence*：10 题，每题一个作答框 + 一个备注框（可显示参考答案）。
    - 模块2 *Write an Email*：大文本框 + **7 分钟**计时器 + 字数统计。
    - 模块3 *Academic Discussion*：大文本框 + **10 分钟**计时器 + 字数统计。
    - 计时器初始为绿色显示剩余时间，归零后变红并继续累计超时；含 **开始 / 暂停停止 / 重置** 按钮。
  - **批改**：阅读、听力按答案自动批改，显示得分、正确答案、对错标记。
  - **缺少题目按钮**：当答案文档缺题时，可在阅读/听力各模块「补充缺失题目」（仅记录、不计入自动批改）。
  - **提交并导出答案**：导出一个 Excel（含 汇总 / 阅读 / 听力 / 写作 四个表）。
- **做题记录页**：列出本浏览器的提交记录，可一键导出全部记录的 Excel。

## 重新生成题库

见 `DEPLOY.md`「四、以后更新」。简而言之：

```bash
pip3 install python-docx pdfplumber pyspellchecker
python3 parse_answers.py "答案文档所在文件夹"
```

## 关于个别答案文档

少数 2026 系列真题 PDF 的**填空答案在源文件里本身就是残缺的**（只显示词尾片段，如 “gations”）。
网站会自动识别这些卷子，对其填空题**只记录作答、不自动批改**（选择题不受影响），并在页面上提示。
其余卷子的填空、选择、听力均正常批改。
